Compute the CRC of a chunk of data.

Do not include trailing zeroes as a room for the CRC (if there are, then truncate the 'size' parameter so they are not counted)

Check a chunk of data with a trailing CRC.

You have to count the bytes taken by the CRC in the 'size' parameter It performs a division of the entire message by the polynomial. If there is no remainder, the message should not be corrupted (at least by a detectable error).

Parameters
dataPointer to the data memory area
sizeSize of the data parameter in bytes
Returns
The remainder of the division
